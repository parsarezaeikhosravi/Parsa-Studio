## ویژه اجرا بر روی لپ‌تاپ شخصی شما (ویندوز و مک)

پارسا جان، اصلاً نگران نباش! تمام طراحان و برنامه‌نویسان ارشد دنیا روز اولی که می‌خواستند پروژه‌ای را از گیت‌هاب نصب کنند، دقیقاً همین حس را داشتند. من این راهنما را به قدری ساده و مینی‌مال نوشته‌ام که بدون نیاز به هیچ دانش قبلی، بتوانی این سیستم فوق‌پیشرفته را روی کامپیوترت بالا بیاوری.

## 🛠️ مرحله اول: دانلود و نصب ۳ ابزار پیش‌نیاز (فقط برای بار اول)

قبل از اجرای هر کدی، کامپیوتر شما باید سه برنامه پایه را داشته باشد تا زبان کدهای گیت‌هاب را بفهمد. روی هر لینک کلیک کن و آن‌ها را مثل برنامه‌های معمولی ویندوز نصب کن:

1.  **دانلود برنامه Node.js (نود جی‌اس):**
    *   *چیست؟* موتور اجراکننده کدهای جاوااسکریپت روی سیستم شما.
    *   *لینک دانلود:* به سایت **`nodejs.org`** برو و نسخه **LTS** (دکمه سمت چپ) را دانلود و نصب کن. (در زمان نصب، تیک گزینه Add to PATH را حتماً بزن).
2.  **دانلود برنامه Git (گیت):**
    *   *چیست؟* ابزار مخصوص دانلود خودکار پروژه‌ها از گیت‌هاب.
    *   *لینک دانلود:* به سایت **`git-scm.com`** برو و نسخه مخصوص سیستم‌عامل خودت (ویندوز یا مک) را دانلود و نصب کن. (در تمام مراحل نصب فقط دکمه Next را بزن).
3.  **دانلود برنامه VS Code (اگر نصب نداری):**
    *   *لینک دانلود:* به سایت **`code.visualstudio.com`** برو و نصبش کن.

## 💻 مرحله دوم: باز کردن محیط فرمان سیستم (Terminal / PowerShell)

حالا باید محیط سیاه فرمان کامپیوترت را باز کنی تا دستورات را به سیستم بدهی:

*   **در ویندوز (Windows):** کلیدهای **Win + S** را روی کیبورد بزن، کلمه **`PowerShell`** را سرچ کن. روی آن راست‌کلیک کن و گزینه **Run as Administrator** (اجرا به عنوان مدیر) را بزن تا باز شود.
*   **در مک (Mac):** کلیدهای **Command + Space** را بزن، کلمه **`Terminal`** را سرچ و باز کن.

## 🚀 مرحله سوم: دانلود پروژه از گیت‌هاب (Cloning)

حالا کدهای زیر را یکی‌یکی کپی کن، در پنجره پاورشل/ترمینال پیست کن و کلید **Enter** را بزن:

```bash
# ۱. دانلود خودکار کل کدهای پروژه ECC از گیت‌هاب روی سیستم شما
git clone https://github.com/affaan-m/ECC.git

# ۲. ورود به پوشه‌ای که دانلود شده است
cd ECC
```

## 📦 مرحله چهارم: نصب پکیج‌ها و کتابخانه‌ها (Installation)

پروژه‌های مدرن برای اجرا نیاز به یک‌سری کتابخانه فرعی دارند که همه در فایلی به نام `package.json` لیست شده‌اند. دستور زیر را وارد کن تا کامپیوترت خودکار همه‌چیز را از اینترنت دانلود کند:

```bash
# ۳. نصب تمام پکیج‌های فنی پروژه (کمی صبر کن تا دانلود تمام شود)
npm install
```

---

## 🛡️ مرحله پنجم: نصب نهایی غریزه‌ها و کدهای رفتاری عامل‌ها

حالا متناسب با سیستم‌عامل خودت، دستورات زیر را کپی و پیست کن تا عامل‌های هوشمند به ادیتورهای تو (مثل VS Code یا Cursor) متصل شوند:

### الف) اگر سیستم‌عامل تو ویندوز (PowerShell) است:
این دستورات را در پاورشل پیست کن و Enter را بزن:
```powershell
# ۱. ساخت پوشه مخفی قوانین در سیستم شما
New-Item -ItemType Directory -Force -Path "$HOME/.claude/rules/ecc"

# ۲. کپی کردن غریزه‌ها به پوشه کدهای رفتاری
Copy-Item -Recurse rules/common "$HOME/.claude/rules/ecc/"
Copy-Item -Recurse rules/typescript "$HOME/.claude/rules/ecc/"

# ۳. اجرای اسکریپت نصب کامل عامل‌ها روی سیستم شما
.\install.ps1 --profile full
```

### ب) اگر سیستم‌عامل تو مک یا لینوکس (Terminal) است:
این دستورات را در ترمینال پیست کن و Enter را بزن:
```bash
# ۱. ساخت پوشه مخفی قوانین
mkdir -p ~/.claude/rules/ecc

# ۲. کپی کدهای رفتاری
cp -R rules/common ~/.claude/rules/ecc/
cp -R rules/typescript ~/.claude/rules/ecc/

# ۳. اجرای اسکریپت نصب نهایی
./install.sh --profile full
```

**تبریک می‌گویم!** تو همین الان سخت‌ترین کار دواپس دنیا را با دستهای خودت با موفقیت انجام دادی. سیستم ECC با تمام عامل‌های هوشمندش روی کامپیوترت مستقر شد. حالا هر زمان برنامه Cursor یا VS Code را باز کنی، این ربات‌های هوشمند در نوشتن کدهای فوق‌سریع و بدون باگ در کنارت خواهند بود.
